# School Trip Management System

## Description

This project is a  system for managing a school trip.
It allows teachers to manage students and view their real-time locations on a map.

---

## Technologies

### Backend:

* Node.js
* Express
* Socket.IO
* MongoDB

### Frontend:

* React
* React Leaflet
* Socket.IO Client

---

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/brachi0525/trip-management-system
```

### 2. Install dependencies

#### Backend:

```bash
cd backend
npm install
```

#### Frontend:

```bash
cd fronted
cd trip-app
npm install
```

---

## Running the Project

### Start Backend:

```bash
npm run dev
```

### Start Frontend:

```bash
npm run dev
```
